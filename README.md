# 🧮 Smart Calculator

A menu-driven command-line calculator developed in Python as part of the **CodSoft Python Programming Internship**.

This calculator performs basic and advanced mathematical operations with a simple, user-friendly interface.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔢 Power Calculation
- √ Square Root
- 📊 Percentage Calculation
- 📜 Calculation History
- 🗑️ Clear History
- ⚠️ Error Handling (Division by Zero & Invalid Input)
- 🎨 Interactive Menu
- 📅 Welcome Screen with Date & Time

---

## 🛠️ Technologies Used

- Python 3
- Math Module
- Datetime Module
